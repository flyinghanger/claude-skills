# 10 维评分标准

每篇文章在发布前必须经过 10 个维度的评分，每项 0-5 分，满分 50 分。

- **≥ 35 分**：可发布
- **30-34 分**：需修改后重新评分
- **< 30 分**：需重写

## 1. 标题力

> 标题像发微信给朋友，不套路。"答案 + 悬念"结构。

| 分数 | 标准 |
|------|------|
| 5 | 一看就想点，但点进去不会觉得被骗。有信息量又有悬念 |
| 4 | 有吸引力，但可以更精炼或更有信息量 |
| 3 | 中规中矩，能概括内容但没有点击欲望 |
| 2 | 太平淡或太标题党 |
| 1 | 让人不想点，或点进去发现完全不符 |
| 0 | 无标题 / 完全无关 |

## 2. 开头力

> 三秒内制造认知冲突。不要套话。

| 分数 | 标准 |
|------|------|
| 5 | 开头即冲突，读完第一段就想继续 |
| 4 | 有冲突但不够锐利，或稍慢了半拍 |
| 3 | 平铺直叙地进入主题，不差但不惊艳 |
| 2 | 开头是套话或背景介绍，需要耐心才能读下去 |
| 1 | 开头让人想关掉 |
| 0 | 无开头 / 完全跑题 |

## 3. 人味

> 有个人经历、观察、判断。不是二手信息综述。

| 分数 | 标准 |
|------|------|
| 5 | 有具体场景、有私人细节、有立场，读起来像真人在讲故事 |
| 4 | 有个人视角，但某些段落回到了"综述模式" |
| 3 | 有零星的个人经历点缀，整体偏信息整理 |
| 2 | 几乎全是二手信息，偶尔加一句"我觉得" |
| 1 | 完全是信息搬运，没有任何个人痕迹 |
| 0 | 机器生成感明显 |

## 4. 信息密度

> 每段都在推进论证，没有水段。

| 分数 | 标准 |
|------|------|
| 5 | 每段都有新信息或新推进，删掉任何一段都会觉得缺了什么 |
| 4 | 整体紧凑，有 1-2 处可以精炼 |
| 3 | 有几段在重复已经表达的观点，或填充了不必要的过渡 |
| 2 | 明显有"凑字数"的段落 |
| 1 | 大量冗余，核心内容可以压缩到一半 |
| 0 | 几乎没有实质内容 |

## 5. 跨域类比

> 用 A 领域的直觉解释 B 领域的问题。

| 分数 | 标准 |
|------|------|
| 5 | 有多个跨域类比且全部精准，读者会产生"原来如此"的顿悟 |
| 4 | 有跨域类比且有效，但可以更精准或更多 |
| 3 | 有类比但比较常见（如"像乐高积木一样"），缺少惊喜 |
| 2 | 类比牵强或不准确 |
| 1 | 试图类比但完全失败 |
| 0 | 没有任何类比 |

## 6. 结构

> 全文有清晰的叙事弧线，不是观点的平铺罗列。

| 分数 | 标准 |
|------|------|
| 5 | 结构本身就在讲故事，节奏有起伏，读者被引导着走完全程 |
| 4 | 结构清晰合理，但节奏可以更好 |
| 3 | 有章节划分，但更像"列表"而非"叙事" |
| 2 | 结构混乱，读到中间不知道在讲什么 |
| 1 | 没有明显结构 |
| 0 | 完全混乱 |

## 7. 闭环

> 结尾回应开头。读完有"圆了"的感觉。

| 分数 | 标准 |
|------|------|
| 5 | 结尾精准回应开头的问题/场景/悬念，读者会有"啊，绕回来了"的满足感 |
| 4 | 有回应但不够精准，或回应方式可以更巧妙 |
| 3 | 结尾和开头有关联，但不是刻意设计的闭环 |
| 2 | 结尾另起话题，和开头脱节 |
| 1 | 戛然而止，没有收尾 |
| 0 | 没有结尾 |

## 8. Greatest Hit

> 全文有没有一句话值得被单独摘出来传播。

| 分数 | 标准 |
|------|------|
| 5 | 有一句（或几句）读完会想截图分享的金句，且是原创的洞察而非陈词滥调 |
| 4 | 有值得记住的表述，但还差一点"传播力" |
| 3 | 有不错的总结性语句，但不够锐利 |
| 2 | 没有特别想记住的句子 |
| 1 | 全文表述平平 |
| 0 | 表述有问题 |

## 9. Fact-check

> 所有可核查的声明是否都经过验证。

| 分数 | 标准 |
|------|------|
| 5 | 所有数据、引用、人名、因果关系都经过独立核查，零错误 |
| 4 | 绝大部分已核查，有 1-2 处小的不确定（非关键信息） |
| 3 | 核心数据已核查，但部分边缘信息未验证 |
| 2 | 有明显的事实错误但不影响核心论点 |
| 1 | 多处事实错误，影响可信度 |
| 0 | 大量错误或完全未核查 |

## 10. 参考文献

> 引用是否充分、规范、可追溯。

| 分数 | 标准 |
|------|------|
| 5 | 引用充分（≥5 条），格式规范，包含一手来源（论文、官方文档），全部可追溯 |
| 4 | 引用充分，格式规范，但一手来源比例可以更高 |
| 3 | 有引用但不够充分，或格式不统一 |
| 2 | 引用很少，或大部分是二手来源 |
| 1 | 几乎没有引用 |
| 0 | 零引用 |
